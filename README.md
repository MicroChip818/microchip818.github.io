# Code Outside the Box
Code Outside the Box is a coding blog where I share projects, review exercises, and document my learning experience in programming.

The goal of this blog is to help others learn programming in a way that is practical and easy-to-understand.

## About
This site is built using Github Pages and Jekyll, with its theme using custom HTML and CSS.

## Status
This project is actively being developed. Expect frequent changes in the website's content.
